Web design has changed a lot in 20 years

A lot has changed since I started doing HTML back in the mid-1990s. Back then, CSS was brand new. Tables were used to do design. And Perl was still being taught in college.

Now the challenge of creating a website for multiple platform (with drastic differences for PCs and phones) is taken to the extreme. In 1996, we just had to account for various screen resolutions. 1200 pixels wide seemed huge back then.

This website uses Bootstrap to help in look right on multiple platforms. This is my first attempt at using Bootstrap, so there are still some bugs to be worked out, but it does make creating a web design very efficient once you get past the learning curve.

It almost makes things too easy to create a nice looking website. So many sites look alike now because of it too. I almost long for the days of animated .gifs, comic sans and tacky background textures.

Minecraft brings a family together

My two kids had been playing Minecraft for a while before I finally joined them.

As a parent, we often look for things to do with our kids that are not mind-numbing.

For example, shows on television can vary from completely inappropriate to "Caillou"-level drudgery.

Minecraft is something that can be fun for all age groups, and it can be educational and provide a creative outlet. Many of the maps on this website were created with help from my kids. If there is a random rainbow-colored building that looks a little out of place, that's their handiwork.
