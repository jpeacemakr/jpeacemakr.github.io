The power of Command Blocks

Minecraft is a massively popular game that is accessible through a PC, tablet, smart phone or even game consoles. But the game is not just mindless fun; it can be used to teach basic principles of coding and logic while also letting kids express themselves creatively. This website teaches the basics of command blocks for the Bedrock Edition. More complex commands can be done with the Java Edition of Minecraft, but the Bedrock version is more accessible across multiple platforms. This site was created by James Peacemaker as a class project at the University of Mary Washington in Fredericksburg, Va. This website is not affiliated with the makers of Minecraft.

The tutorial

Command blocks can be surprisingly powerful in Minecraft Bedrock Edition. Learn about the different types of command blocks and how to chain them together to create more complex instructions. You can add a whole new level of creativity to your maps.
