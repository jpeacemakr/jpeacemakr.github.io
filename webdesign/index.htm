Websites

Don't Torpedo the Whales: This is a game using the html canvas element. It uses two overlapping canvases controlled by javascript. It creates the game board based on the size of the screen. It also uses some jQuery, original .svg artwork and sound effects.

Minecraft Command Block guide: This website teaches people how to use command blocks in Minecraft Bedrock Edition. It has a responsive web design for PC, phone or tablet using Bootstrap.

Job / Intership Board prototype: This is a prototype for a job board website. This design was mine and was later implemented by a team in our software engineering class. It used Next.js (a REACT framework) for the front end and custom Javascript APIs (using Node.js) and a PostgreSQL database for the back end.

The Capital newspaper prototype: A prototype design for The Capital newspaper website circa early 2000s. This is a bit outdated.
